Glitch Image Effect README

=== Intro ===

You can get a fullscreen camera image effect with this asset. Please goto AssetStore to wathcing the demo video.

The public link on AssetStore is https://assetstore.unity.com/packages/slug/123680

=== How to use ===

1. Add the Glitch Image Effect Script to your Main Camera.

2. Play scnene, you will see the fullscreen Glitch image effect.

3. There are some custom variable that you can set as your like to get different effects.

=== Compatiability === 

We have test this on PC, MAC, IOS and Android. Each platform works well and get good performance.
